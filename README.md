# 표현식, 문
값: 표현식이 평가되어 생성된 결과 
리터럴: 사람이 이해할 수 있는 문자 또는 약속된 기호를 사용해 값을 생성하는 표기법
표현식: 값으로 평가될 수 있는 문. 표현식이 평가되면 새로운 값을 생성하거나 기존값을 참조
문: 프로그램을 구성하는 기본단위이자 최소실행
토큰: 문법적으로 더이상 쪼갤 수 없는 코드의 기본 요소

세미콜론을 사용하여 문의 종료를 표현하지만 자바스크립트에는 자동삽입 기능이 있다

표현식인 문과 아닌것을 구분 하려면 변수에 할당해 보면 된다
할당문은 그 자체가 표현식이지만 완전한 문 이기도 한다

# 데이터 타입
1.숫자
정수,실수,음의 정수,2진수,8진수,16진수 를 넣을 수 있음
숫자 타입은 모도 정수로 취급 한다(1 == 1.0) 
자바스크립트에 있는 세가지 특별한 숫자 타입: Infinity, -Infinity, NaN

2.문자열
큰 따온표, 작은 따옴표, 백틱(``)으로 문자열의 범위를 표시함
큰(작은)따옴표 뒤의 큰(작은)따옴표도 문자열 취급 한다

3.템플릿 리터럴
멀티라인 문자열,표현식 삽입,태그드 템플릿릿을 삽입 할 수 있게 해주는 데이터 타입
백틱을 이용하여 표현
멀티라인 문자열:엔터키로 줄바꿈을 해서 써도 됨
표현식 삽입:${변수}로 변수를 쉽게 삽입 할 수 있다(파이썬 f스트링 느낌)

4.불리언 타입
true, false로 되어있고 조건문에서 쓰임

5.undefined 타입
var로 선언한 변수의 초기값은 undefined 
변수가 이후로 선언된 값을 할당 할 때 까지 남아있음

6.null 타입
변수의 값이 없다는 것을 의도적으로 명시
변수가 참조하던 이전에 값을 명시적으로 제거하여 이제는 참조하지 않겠다는 뜻

7.객체타입 
변경 불가능한 원시 타입의 값. 다른 값과 중복되지 않는 유일무이한 값이다.

자바스크립트는 변수에 값을 할당하면서 알아서 변수의 타입을 결정함(동적 추론)

# 연산자
연산자와 피연산자로 구성
피연산자는 값으로 평가될 수 있는 표현식이여야함 
피연산자가 "값"이면 연산자는 피연산자로 새로운 값을 만드는 역할
